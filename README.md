# ClassPAL
OpenSource Tuition Class Management System. Complete Web-Based Tuition Class Management System for Tuition Class Teachers.
Responsive User Interface.

Features
ClassPAL will run over all kinds of devices seamlessly.
Add Unlimited Students.
You can add any amount of students and their details and view quickly.
Add Unlimited Payments.
You can add unlimited student payments & print receipt quickly.
Mark Student Attendance.
You can add student attendance and absents easily.
Manage Multiple Subjects.
Manage multiple subjects in one place easily.
All in one Report Dashboard.
Monitor status of your classes from all in one report dashboard.
Parent Monitor.
Parents able to view child, attendance, payments & class announcements any time anywhere from any devices.

Minimum System Requirements
PHP Version 7.4
MySQL Version 5.x or higher
