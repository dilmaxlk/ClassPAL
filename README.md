<p><span style="font-size:36px"><strong>ClassPAL</strong></span><br />
<strong>OpenSource Tuition Class Management System. Complete Web-Based Tuition Class Management System for Tuition Class Teachers.<br /></strong></p>

<p><br />
<strong><span style="font-size:20px">Features</span></strong></p>

<ul>
	<li><strong>Responsive User Interface</strong></li>
	<li><strong>ClassPAL will run over all kinds of devices seamlessly.</strong></li>
	<li><strong>Add Unlimited Students.</strong></li>
	<li><strong>You can add any amount of students and their details and view quickly.</strong></li>
	<li><strong>Add Unlimited Payments.</strong></li>
	<li><strong>You can add unlimited student payments &amp; print receipt quickly.</strong></li>
	<li><strong>Mark Student Attendance.</strong></li>
	<li><strong>You can add student attendance and absents easily.</strong></li>
	<li><strong>Manage Multiple Subjects.</strong></li>
	<li><strong>Manage multiple subjects in one place easily.</strong></li>
	<li><strong>All in one Report Dashboard.</strong></li>
	<li><strong>Monitor status of your classes from all in one report dashboard.</strong></li>
	<li><strong>Parent Monitor.</strong></li>
	<li><strong>Parents able to view child, attendance, payments &amp; class announcements any time anywhere from any devices.</strong></li>
</ul>
